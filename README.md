# Curso-de-Fundamentos-de-Desarrollo-Web

## Resumen del Curso

Este curso de 1 hora introduce a estudiantes de secundaria a la creación de sitios web sencillos utilizando HTML y CSS. El contenido está diseñado para estudiantes entre 12 y 17 años que no tienen experiencia previa en programación.

**¡No requiere instalaciones!** Todas las actividades se realizan desde el navegador utilizando herramientas gratuitas en línea.

### Información General

* **Duración total:** 60 minutos
* **Público objetivo:** Estudiantes de 12 a 17 años
* **Prerrequisitos:** Ninguno
* **Herramientas necesarias:** Navegador web (Chrome, Firefox, Edge o Safari)
* **Editor online principal:** CodePen
* **Repositorio de código fuente:** [https://github.com/Master-Squad/webdev-course-master-squad.git](https://github.com/Master-Squad/webdev-course-master-squad.git)

# Secuencia de Lecciones

## Lección 1: ¿Qué es el Desarrollo Web? (5 minutos)

### Descripción

Aprende qué es una página web y cómo HTML y CSS trabajan juntos para construir sitios web modernos.

### Objetivos

* Comprender qué es una página web.
* Diferenciar HTML y CSS.
* Conocer el entorno de trabajo en línea.

### Video

https://youtube.com/[video1]

### Actividad práctica

https://codepen.io/[lesson1]

### Consejos clave

* HTML = estructura.
* CSS = apariencia.

---

## Lección 2: Introducción a HTML (10 minutos)

### Descripción

Aprende la estructura básica de una página HTML y crea tu primera página web.

### Objetivos

* Comprender etiquetas y elementos.
* Utilizar encabezados y párrafos.
* Construir una página HTML básica.

### Video

https://youtube.com/[video2]

### Actividad práctica

https://codepen.io/[lesson2]

### Consejos clave

* Utiliza `<html>`, `<head>` y `<body>`.
* Los títulos se crean con `<h1>`.
* Los párrafos se crean con `<p>`.

---

## Lección 3: Elementos HTML Comunes (12:44 minutos)

### Descripción

Agrega listas, imágenes y enlaces para enriquecer tu página web.

### Objetivos

* Insertar imágenes.
* Crear listas.
* Agregar enlaces.

### Video

[Haz click aquí para ver la Lección 3](https://youtu.be/hASkIcAW17s)👈

### Actividad práctica

[Click aquí para ver el proyecto](https://codepen.io/braden-g-the-vuer/pen/zxNNNpa)👈Guarda y comparte

### Consejos clave

* `<img>` para imágenes.
* `<ul>` y `<li>` para listas.
* `<a>` para enlaces.

## Lección 4: Introducción a CSS (6:35 minutos)

### Descripción

Aprende a cambiar colores, fuentes y apariencia de una página web.

### Objetivos

* Comprender selectores.
* Aplicar estilos básicos.
* Modificar colores y fuentes.

### Video

[Haz click aquí para ver la Lección 4](https://youtu.be/nwD0cCbK1ko)👈

### Actividad práctica

[Click aquí para ver el proyecto](https://codepen.io/yqbhbtod-the-animator/pen/gbgggKa)👈Guarda y comparte
